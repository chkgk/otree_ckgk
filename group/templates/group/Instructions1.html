Sie beginnen nun mit dem zweiten Teil des Experiments. Wir machen Sie zunächst mit den beiden Rollen dieses Studienteils vertraut: Kunden und Finanzberater.

Kunden

In der Rolle des Kunden besitzen Sie {{ Constants.endowment_principals }}, die investiert werden sollen. Sie treten daher in Kontakt mit einem Finanzberater, dem Sie mitteilen, wie die Anlageentscheidung für Sie getroffen werden soll. Dazu wählen Sie eine der fünf Anlagestrategien aus, die Sie bereits in Teil 1 kennengelernt haben. Am Ende erhalten sie als Kunde das Ergebnis der für Sie getroffenen Investition als Auszahlung. Sie können Ihrem Finanzberater dann eine von vier vorgegebenen Nachrichten schicken, in der Sie Ihre Zufriedenheit oder Unzufriedenheit mit seiner Entscheidung ausdrücken.

Finanzberater

In der Rolle des Finanzberaters treffen Sie die Investitionsentscheidung für fünf Kunden. Sie erhalten dazu von jedem Ihrer Kunden eine Mitteilung, welche Anlagestrategie verfolgt für ihn werden soll. Sie sind als Finanzberater jedoch nicht an diese Mitteilungen gebunden, sondern entscheiden frei. {% if player.compensation == "fixed" %} Am Ende erhalten Sie als Finanzberater einen fixen Betrag in Höhe von {{ Constants.fixed_payment }} als Auszahlung. {% elif player.compensation == "variable_result" %} Nach der Investitionsentscheidung wird einer der Kunden für Sie auszahlungsrelevant. Als Finanzberater erhalten Sie einen fixen Betrag in Höhe von {{ Constants.fixed_payment }} und einen prozentualen Anteil am Investitionsergebnis des ausgewählten Kunden als Auszahlung. {% elif player.compensation == "variable_profit" %} Nach der Investitionsentscheidung wird einer der Kunden für Sie auszahlungsrelevant. Als Finanzberater erhalten Sie einen fixen Betrag in Höhe von {{ Constants.fixed_payment }} und einen prozentualen Anteil am positiven Ertrag des ausgewählten Kunden als Auszahlung. {% endif %} Sie erhalten die von dem auszahlungsrelevanten Kunden ausgewählte Nachtricht zu dessen Zufriedenheit bzw. Unzufriedenheit mit Ihrer Entscheidung.
