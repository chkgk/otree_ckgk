Zu Beginn jeder Runde sehen Sie die Lotterie, die Ihnen zunächst zugeteilt wurde. Dies ist Ihre Anfangslotterie. Wenn Sie diese Anfangslotterie behalten möchten, müssen Sie nichts tun und lediglich {{ interval }} Sekunden warten. Die Runde endet dann und Ihre Anfangslotterie wird gespielt.

{% if default == "Safe" %}

Alternativ können Sie die Ergebnisse der Anfangslotterie ändern, indem Sie auf den Button „Anpassen“ klicken. Dazu haben Sie {{ interval }} Sekunden Zeit. Sobald Sie auf den Button „Anpassen“ klicken, wird vom grünen Balken (bzw. Ergebnis) der Betrag abgezogen der unter dem Balken angezeigt wird (im Beispiel -1.00) und zum gelben Balken (bzw. Ergebnis) der Betrag hinzugezählt, der unter dem Balken angezeigt wird (im Beispiel +1.50). D.h. der Unterschied in den Auszahlungen wird größer, allerdings bekommen Sie auch einen größeren Betrag, wenn das „gelbe“ Ergebnis ausgewählt wird.

Das folgende Beispiel soll verdeutlichen, wie das Anpassen einer Lotterie funktioniert. Die Anfangslotterie wird Ihnen im linken Bild dargestellt. Sie erhalten 8 Euro, wenn das „grüne“ Ereignis und 13 Euro, wenn das „gelbe“ Ereignis gezogen wird. Im rechten Bild sehen Sie die Lotterie, die entstehen würde, wenn Sie auf „Anpassen“ drücken. Über dem grünen Balken steht jetzt die Zahl 7 und über dem gelben Balken die Zahl 14.50. Die beiden Zahlen unter den Balken geben an wie sich die jeweiligen Auszahlungen ändern wenn Sie „Anpassen“ drücken. Diese Zahlen unterscheiden sich von Runde zu Runde. Es ist allerdings immer der Fall, dass sich die Auszahlungen des grünen Ereignisses verringern und des gelben Ereignisses erhöhen.

Wenn Sie auf „Anpassen“ geklickt haben, wird Ihnen die neue Lotterie nach Ablauf der {{ interval }} Sekunden angezeigt. Danach haben Sie wieder die Möglichkeit zu entscheiden, ob Sie diese neue Lotterie behalten wollen oder ob Sie einen weiteren Anpassungsschritt vornehmen möchten. Wollen Sie die neue Lotterie behalten, müssen Sie wiederum {{ interval }} Sekunden warten. Die Lotterie wird dann automatisch ausgewählt und eines der Ereignisse (grün oder gelb) wird zufällig ausgewählt. Sofern Sie die neue Lotterie erneut anpassen wollen, klicken Sie bitte erneut auf „Anpassen“. Sie haben dazu wieder {{ interval }} Sekunden Zeit. Sie könne die Lotterie so lange anpassen, bis der grüne Balken (Ergebnis) 0 Euro erreicht. In diesem Fall wählen Sie diese Lotterie.
